# E-Commerce Inventory & Order Management System

## Overview
This project implements an **E-Commerce Inventory Management System** in Java to manage products, customers, orders, and reviews.  
All data structures are implemented manually (no Java Collections).  
The system includes time and space complexity analysis for each main function.

## Features
- Add, remove, update, and search products  
- Track out-of-stock items  
- Register customers and manage orders  
- Add and edit reviews  
- Calculate average product ratings  
- Retrieve top 3 products by rating  
- List orders between specific dates  
- Find common reviewed products (rating > 4) between two customers  
- Read and process data from CSV files
